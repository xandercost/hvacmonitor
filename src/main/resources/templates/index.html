HVAC Monitoring Service

To view reports for a given date range edit the url using the format: http://localhost:8080/hvac_data/reports/start={start}/end={end} where start and end are dates formatted as MM-dd-yyyy.
To view June results copy the following: http://localhost:8080/hvac_data/reports/start=06-01-2020/end=06-30-2020
To view July results copy the following: http://localhost:8080/hvac_data/reports/start=07-01-2020/end=07-31-2020
